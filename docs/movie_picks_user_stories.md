# Movie Picks — User Stories

## Authentication
- As a user, I want to create an account so my watchlist is private and saved to my profile.
- As a user, I want to log in and out securely so only I can access my watchlist.

## Movie Search
- As a user, I want to search for movies by title in real time so I can quickly find what I'm looking for.
- As a user, I want to see movie posters, year, and genre in search results so I can identify the right film.
- As a user, I want to select a movie from search results so the details are pre-filled automatically.

## Watchlist Management
- As a user, I want to add a movie to my watchlist so I can keep track of films I want to see.
- As a user, I want to view all movies in my watchlist so I can browse what I've saved.
- As a user, I want to view details for a single movie so I can see the plot, cast, and ratings.
- As a user, I want to remove a movie from my watchlist so I can keep it organized.

## Watch Status & Ratings
- As a user, I want to mark a movie as watched so I can track what I've already seen.
- As a user, I want to rate a movie 1–10 after watching so I can record my opinion.
- As a user, I want to add personal notes to a movie so I can write a mini-review or reminder.

## Filtering & Sorting
- As a user, I want to filter my watchlist by watched/unwatched so I can focus on what's next.
- As a user, I want to sort my watchlist by rating or date added so I can find movies easily.
